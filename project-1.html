WEB APPLICATION DEVELOPMENT AND DEPLOYMENT WITH CI/CD

This website is a robust platform designed to deliver an integrated media experience, allowing users to seamlessly access Spotify and YouTube content. Developed using HTML, CSS, and JavaScript, it leverages open-source code from GitHub. The platform features a Dockerized environment, published on Docker Hub, and utilizes a CI/CD pipeline with Docker, Jenkins, and Terraform for automated testing and deployment, ensuring high availability on AWS. Version control is maintained through GitHub, and infrastructure is managed using Terraform, with performance optimizations implemented for an optimal user experience.
